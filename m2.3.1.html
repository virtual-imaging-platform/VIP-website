VIP milestone M2.3.1 - user interface to execution environment

A web portal has been developed to provide an interface to the execution environment for image simulation.

This portal is available at http://vip.creatis.insa-lyon.fr

For now anonymous users can only see the front page. Users have to request a grid certificate and register to the biomed VO to get access to the simulators.

The portal is currently at release 0.3.3. The development roadmap, including new features planned until the end of the project, software releases and SVN repository is available at http://vip.creatis.insa-lyon.fr:9002/projects/platform/roadmap
